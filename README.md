# spring-boot-starter-pf4j2
pf4j starter for spring boot

### 说明


 > 基于 Pf4j 的 Spring Boot Starter 实现

1. 整合Pf4j插件能力

### Maven

``` xml
<dependency>
	<groupId>${project.groupId}</groupId>
	<artifactId>spring-boot-starter-pf4j2</artifactId>
	<version>${project.version}</version>
</dependency>
```
